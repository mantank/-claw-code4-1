# xiaolongxia.app RPG像素风改版方案

> 制定时间：2026-03-13
> 风格参考：星露谷物语（Stardew Valley）像素风
> 核心概念：养龙虾 = 经营像素农场，技能是种子，副本是田地，收获是效率提升
> 等旭确认后交004执行

## 一、视觉风格定义

### 1.1 风格关键词

**星露谷物语 × 龙虾养成 × AI工具站**

- 像素风（Pixel Art）：8-bit / 16-bit复古游戏感
- 温暖治愈：不是赛博朋克的冷硬科技感，是农场经营的温馨感
- 成就驱动：每学一个技能、每通一个副本都有"叮！"的满足感
- 简洁实用：像素风是外衣，内容实用性是核心

### 1.2 色彩方案

参考星露谷物语的调色盘，暖色为主：

```
主色调：
  - 草地绿 #4CAF50 / #66BB6A（主背景、按钮）
  - 泥土棕 #8D6E63 / #A1887F（卡片边框、面板底色）
  - 天空蓝 #64B5F6 / #90CAF9（顶栏、链接）

强调色：
  - 龙虾红 #EF5350 / #E53935（龙虾角色、重要标记）
  - 金币黄 #FFD54F / #FFC107（星级、成就、高亮）
  - 经验紫 #AB47BC / #9C27B0（经验值、进阶内容）

中性色：
  - 面板米色 #FFF8E1 / #FFFDE7（内容区背景）
  - 像素黑 #212121（文字、边框）
  - 阴影灰 #9E9E9E（次要文字）

禁止使用：
  - 渐变色（破坏像素感）
  - 高饱和度霓虹色
  - 纯白大面积背景（太现代，不像素）
```

### 1.3 字体

```
标题/导航：像素字体
  - 推荐：Zpix（中文像素字体，开源免费）
  - 备选：丁卯点阵体（中文像素）
  - 英文：Press Start 2P（Google Fonts免费）

正文/内容：现代中文字体（保证可读性）
  - 推荐：思源黑体 / Noto Sans SC
  - 正文不用像素字体（长文阅读太累）

规则：
  - 标题、导航、按钮、标签 → 像素字体
  - 正文、教程内容、代码块 → 现代字体
  - 两种字体混搭 = 像素风外衣 + 实用内核
```

### 1.5 像素角色 & 图标

需要制作的像素角色/图标素材：

```
核心角色：
  🦞 龙虾本虾 — 站立/行走/学习/战斗 4个状态
     - 站立：默认态，首页展示
     - 学习：翻书动作，秘籍页展示
     - 战斗：举钳子，副本页展示
     - 升级：头顶冒星星，成就展示

场景图标（32x32像素）：
  ⛏️ 内容矿洞 — 矿洞入口+镐子
  ⚡ 效率竞技场 — 竞技场/擂台
  🏡 日常任务 — 小房子/邮箱
  🔥 代码熔炉 — 炼炉/铁砧
  🕵️ 情报暗哨 — 望远镜/暗影
  🌱 新手村 — 小树苗/路标

物品图标（16x16像素）：
  📜 秘籍书 — 发光的书本
  ⚔️ 副本入口 — 传送门/洞穴口
  🎒 套装包 — 背包/宝箱
  ⚠️ Boss — 小怪物头像
  🏆 成就 — 奖杯/勋章
  💰 掉落 — 金币/宝石（代表节省的时间）

制作方式：
  - 方案A：用Nano Banana 2生成（prompt指定pixel art style）
  - 方案B：用Aseprite/Piskel在线像素画工具手绘
  - 方案C：找免费像素素材包改色（推荐 itch.io 免费素材）
  - 建议：A生成草稿 → B手动微调
```

### 3.1 CSS像素风框架

```css
/* 全局像素风基础样式 */

/* 像素字体 */
@import url('https://fonts.googleapis.com/css2?family=Press+Start+2P&display=swap');
@font-face {
  font-family: 'Zpix';
  src: url('/fonts/zpix.woff2') format('woff2');
}

/* 像素风变量 */
:root {
  --pixel-green: #4CAF50;
  --pixel-brown: #8D6E63;
  --pixel-blue: #64B5F6;
  --pixel-red: #EF5350;
  --pixel-gold: #FFD54F;
  --pixel-purple: #AB47BC;
  --pixel-cream: #FFF8E1;
  --pixel-black: #212121;
  --pixel-border: 3px solid var(--pixel-black);
  --pixel-shadow: 3px 3px 0px var(--pixel-black);
  --font-pixel: 'Zpix', 'Press Start 2P', monospace;
  --font-body: 'Noto Sans SC', sans-serif;
}

/* 像素风卡片 */
.pixel-card {
  border: var(--pixel-border);
  box-shadow: var(--pixel-shadow);
  background: var(--pixel-cream);
  image-rendering: pixelated;
}

/* 像素风按钮 */
.pixel-btn {
  font-family: var(--font-pixel);
  border: var(--pixel-border);
  box-shadow: var(--pixel-shadow);
  padding: 8px 16px;
  cursor: pointer;
  transition: transform 0.1s;
}
.pixel-btn:active {
  transform: translate(3px, 3px);
  box-shadow: none; /* 按下效果 */
}

/* 像素风进度条 */
.pixel-progress {
  border: var(--pixel-border);
  height: 20px;
  background: #e0e0e0;
}
.pixel-progress-fill {
  height: 100%;
  background: var(--pixel-green);
  image-rendering: pixelated;
  /* 方块状填充效果 */
  background-image: repeating-linear-gradient(
    90deg, transparent, transparent 8px, 
    rgba(0,0,0,0.1) 8px, rgba(0,0,0,0.1) 10px
  );
}

/* 标题用像素字体，正文用现代字体 */
h1, h2, h3, nav, .badge, .btn-text {
  font-family: var(--font-pixel);
  font-size: 0.85em; /* 像素字体通常需要缩小 */
}
body, p, .content {
  font-family: var(--font-body);
}
```

### 3.2 动画效果

```
需要的CSS/JS动画：
1. 龙虾站立呼吸动画 — 上下轻微浮动（CSS animation）
2. 卡片hover效果 — 边框变金色 + 轻微上浮
3. 按钮点击效果 — 下沉3px（去掉shadow）
4. 页面切换 — 像素风转场（黑色方块从四周合拢再展开）
5. 数字滚动 — 统计数字从0滚到目标值
6. 星级动画 — 星星依次点亮（左到右）

不需要的：
- 不要粒子特效（太现代）
- 不要3D变换（破坏像素感）
- 不要毛玻璃/模糊（跟像素风冲突）
```

### 3.3 响应式适配

```
桌面端（>1024px）：
  - 完整像素风布局
  - 副本地图横向展示
  - 卡片3列网格

平板端（768-1024px）：
  - 卡片2列
  - 地图改为列表

手机端（<768px）：
  - 卡片单列
  - 像素字体稍微放大（小屏像素字体太小看不清）
  - 导航改为底部Tab栏（像素风图标）
  - 底部Tab：🏠首页 📜秘籍 ⚔️副本 🌱新手村
```
