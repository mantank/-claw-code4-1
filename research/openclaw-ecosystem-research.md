## 一、别人怎么玩OpenClaw（25+真实用例）

### 🔥 最热门的5个应用场景

**1. 早安简报（Morning Briefing）— 最受欢迎，没有之一**
- 每天早上7-8点自动推送：天气+日历+待办+新闻+健康数据
- 替代5-6个App的早晨打开
- 需要VPS才能稳定每日推送
- ClawHub有现成skill：`morning-briefing`、`morning-briefing-pro`
- **我们的差距**：我们已经有选题日报了，但还没做完整的早安简报。可以整合：天气+日历+选题+待办+昨日复盘，一条消息搞定

**2. 全自动内容流水线（Content Pipeline）**
- 从选题→写作→排版→发布全链路自动化
- 有人用OpenClaw跑整个YouTube频道（分析几百个视频的数据）
- 品牌语气一致性靠SOUL.md控制
- ClawHub: `agent-content-pipeline`、`automated-content-machine`
- **我们的现状**：选题cron已有，但写作→排版→发布还是手动。下一篇文章《AI自动写公众号》就是讲这个

**3. 多Agent商业运作（Business on Autopilot）**
- 有人让AI跑整个公司：邮件+CRM+任务管理+日报
- CEO级仪表盘：多Agent监督
- 自主商业经理：会自己spawn子Agent
- **我们的方向**：正在做的"一人公司"就是这个路线，001+002协作

**4. 手机编程（Code from Phone）**
- 通过Telegram发消息就能写代码、提PR、部署
- 不需要打开电脑
- GitHub skill + Linear skill 配合使用
- **可以考虑**：旭在通勤时通过Telegram给001下指令，001帮他改代码/写文章

**5. 智能家居控制**
- 用自然语言控制家电、灯、温度
- 需要Home Assistant等中间件
- **暂时不适用**：旭还没有智能家居设备

### ⚡ 我们现在没做但值得做的场景

**邮件自动处理**
- 每天自动整理邮箱，分类重要/不重要
- 需要：`email-daily-summary` skill 或 IMAP接入
- 价值：旭不用每天手动翻邮箱

**语音日记**
- 发语音消息→自动转文字→整理成结构化日记
- 每晚9点自动汇总当天所有语音笔记
- 需要：ffmpeg + Whisper
- **特别适合旭**：他说过"听比读效果好"，反过来说比写也更自然

**日历自动管理**
- 自动检测冲突、安排会议、发提醒
- Google Calendar API接入
- 我们目前只在心跳里简单提醒，没有真正接入日历

**微信公众号搜索**
- ClawHub有 `wechat-article-search`、`wechat-search` 
- 可以搜竞品文章、分析对标账号
- **可以给选题cron加一步**：搜微信同类文章看竞品

**小红书自动发布**
- `xiaohongshu-publish`、`xiaohongshu-publisher`
- 自动发布小红书长文+图文
- **可以考虑**：把公众号内容同步分发到小红书

## 二、ClawHub热门Skills推荐（经过安全评估）

### ⭐ 第一梯队：安全+高价值

| Skill | 功能 | 适用场景 | 安全评级 |
|-------|------|---------|---------|
| `morning-briefing-pro` | 早安简报 | 每日信息聚合 | ✅安全 |
| `agent-content-pipeline` | 内容流水线 | 自动化写作发布 | ✅安全 |
| `email-daily-summary` | 邮件日报 | 邮箱自动整理 | ⚠️需专用邮箱 |
| `notion` | Notion集成 | 我们已有脚本，可升级 | ✅安全 |
| `clawrouter` | 智能路由 | 按任务复杂度分配模型，省70%成本 | ✅安全 |
| `wechat-article-search` | 微信文章搜索 | 竞品分析 | ⚠️需评估 |

### ⭐ 第二梯队：有价值但需验证

| Skill | 功能 | 备注 |
|-------|------|------|
| `xiaohongshu-mcp` | 小红书自动化 | 可搜索/发布，需验证安全性 |
| `x-post-automation` | X/Twitter自动发帖 | 配合Grok搜索可以自动发帖 |
| `automation-workflows` | 工作流编排 | 多步骤自动化 |
| `agent-browser` | 浏览器自动化 | 已安装，可做网页操作 |

### ⚠️ 安全提醒（来自社区实测）

- ClawHub 3000+个Skills里，约**15%有可疑行为**（Reddit安全研究）
- 典型恶意行为：
  - "音乐"skill偷偷搜索本地文件里的身份证号
  - "备份"skill把聊天记录发到第三方服务器
- **安装任何新skill前必须检查**：
  1. SKILL.md是否清晰说明功能和权限
  2. 有没有可疑的网络请求
  3. 最近是否有更新/维护
  4. 我们已有 `skill-vetting` skill可以做安全审查

## 三、ClawRouter — 值得立即安装的省钱神器

**功能**：根据任务复杂度自动选择模型
- 简单任务（查天气、翻译）→ 用便宜模型（Haiku/Flash）
- 复杂任务（写文章、分析）→ 用贵模型（Opus/Gemini Pro）
- 社区反馈：成本降低约70%

**对我们的意义**：
- 001目前固定用Opus，所有任务都烧贵模型
- 装了ClawRouter后，简单对话自动切便宜模型
- 特别适合：心跳检查、简单查询、状态报告这些不需要Opus的场景

## 四、我们可以立即行动的3件事

### 1. 安装ClawRouter（节省token成本）
```bash
clawhub install clawrouter
```
配置后简单任务自动走便宜模型，预计省50-70%。

### 2. 升级早安简报
把现有的选题cron扩展为完整早安简报：
- 天气（已有weather skill）
- 今日日程（接Google Calendar）
- 选题推荐（已有）
- 昨日待办进度（从Notion拉）
- 一句话提醒（WgetCloud到期等）

### 3. 加入微信文章搜索
给选题cron加一步：搜微信同类文章，分析竞品标题和话题。
