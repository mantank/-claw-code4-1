# 办公室白板手绘风格

## 风格ID
office-whiteboard

## 风格名称
办公室白板手绘风格

## 风格描述
一套极具真实感的办公室白板手绘风格横幅图，模拟真实白板上用马克笔手绘的效果。整体氛围亲切、直观、接地气且有信息传达力。融合了真实办公场景摄影、手绘简笔画插图和信息图表设计，适合用于社交媒体封面、公众号头图、技术分享配图、教程横幅等场景。不限定生成模型，可搭配任意支持文生图的模型使用。

## 基础提示词模板

你是一位专家级场景摄影与信息图设计师，请生成一张超高清真实办公室白板照片。比例2.35:1横幅构图，白板上的手绘内容必须铺满整个白板面，不允许出现大面积空白区域，每个区域都要有图形或文字元素填充。

全局视觉语言方面，白板占满整个画面100%，四边紧贴画框，不留任何办公室背景、墙面、地板或周围环境。白板为标准办公白板，带有银色金属边框紧贴画面四周，底部托盘上放着红绿蓝黑四色马克笔和板擦。白板表面有轻微的反光光斑和擦不干净的灰色使用痕迹，体现日常使用感。画面构图：白板主体100%充满画幅，无任何背景露出，无四周留白，无办公室场景。

整体亮度要求：画面整体明亮通透，曝光补偿+1.2EV，白板表面洁白明亮，避免偏暗或灰蒙蒙的色调。自然日光充足照射，白板上的文字和图形色彩鲜明饱和。

关于中文文字渲染规范（极其重要）：
1. 白板上所有中文文字必须是正确、清晰、可辨认的简体中文汉字，绝对不允许出现乱码、日文假名、韩文、错别字、缺笔画、多笔画、字形扭曲或无法辨认的字符。
2. 为保证中文渲染质量，白板上的中文文字总量必须严格控制：标题不超过15个汉字，左侧说明文字不超过12个汉字，右侧说明文字不超过12个汉字，底部关键词总共不超过20个汉字。文字越少，每个字的渲染质量越高。
3. 标题字号必须非常大，每个汉字占白板高度的约五分之一，确保字形完整不变形。
4. 英文单词和数字可正常使用，通常比中文渲染更稳定，可优先用英文表达专有名词和品牌名。
5. 如果内容信息量大，优先用简笔画图形表达含义，减少文字依赖。图形比文字更可靠。

关于白板内容绘制规范：
1. 标题文字使用红色超大粗体手写字体，居中放置在白板顶部，具有马克笔书写的笔触质感和轻微的不规则感。
2. 插图必须使用蓝色马克笔简笔画风格，即用简单的线条勾勒轮廓，类似白板会议上随手画的示意图。严禁画成精细插画、水彩风或写实风格。人物用火柴人或极简卡通表现，设备用方框加几根线条表现，保持粗犷随意的手绘感。
3. 强调标记使用绿色马克笔，如对勾、箭头、高亮下划线、圈注等元素。绿色元素要大而醒目。
4. 说明文字和关键词使用黑色马克笔手写，字体较标题小但仍需清晰可读，字号不能太小。
5. 整体版面分区明确，左右两个区域都必须有图形元素（简笔画图标、示意图、框图等），不能只有纯文字。用箭头、连线、框图等元素填充空间，确保视觉饱满。

渲染质量要求：超高清摄影风格，8K分辨率，明亮自然光，浅景深，办公室日光环境，真实材质纹理，画面通透明亮。

## 页面类型模板

### 概念展示页模板
构图逻辑：顶部红色大标题点明主题，标题要占白板宽度的80%以上。白板下方分为左右两区：左侧用蓝色简笔画展示场景（人物、设备、状态等），右侧同样必须有蓝色简笔画图形元素（如手机、电脑、机器人、图表等设备简笔画），配合绿色箭头和文字说明。右侧绝不能只有文字而没有图形。底部黑色手写关键词用竖线分隔，总结核心要素。
使用场景：展示产品概念、工作流自动化、效率提升对比等。

### 流程步骤页模板
构图逻辑：顶部红色大标题说明流程主题。白板中部用蓝色简笔画从左到右依次绘制3至5个步骤节点，每个节点用圆圈或方框包裹一个简笔画图标，节点之间用绿色大箭头连接。每个节点下方用黑色手写标注步骤名称。底部黑色手写补充说明或工具关键词。
使用场景：展示操作流程、使用步骤、工作管线等。

### 对比展示页模板
构图逻辑：顶部红色大标题点明对比主题。白板中间用黑色竖线分为左右两半，左侧标注"Before"或痛点，右侧标注"After"或方案，两侧各用蓝色简笔画配合绿色对勾或红色叉号展示对比内容。底部黑色手写总结关键结论。
使用场景：展示方案对比、前后效果、优劣分析等。

## 技术参数

### 通用配置建议
- 比例：2.35:1（超宽横幅）或 16:9
- 分辨率：2K 或 4K（视模型支持情况选择）
- 不限定生成模型，兼容主流文生图模型

### 适用场景
- 社交媒体封面图、公众号文章头图
- 技术教程配图、产品功能介绍
- 工作流展示、短视频封面
- OpenClaw SK 技能集成

### 色彩规范
- 标题：红色（马克笔粗体）
- 插图：蓝色（马克笔简笔画）
- 强调：绿色（对勾、箭头、高亮）
- 正文/关键词：黑色（马克笔手写）
- 白板底色：明亮洁白，带轻微灰色使用痕迹
- 边框：银色金属

### 中文渲染质量优化策略
| 策略 | 说明 |
|------|------|
| 控制字数 | 全板中文总量控制在60字以内，单行不超过12字 |
| 放大字号 | 标题字号尽可能大，小字也不能低于白板高度的十分之一 |
| 英文替代 | 品牌名、专有名词、缩写优先使用英文（如AI Agent、OpenClaw） |
| 图形优先 | 能用简笔画表达的含义就不用文字，减少文字渲染压力 |
| 分批验证 | 如果一次生成文字不对，可拆成两次：先生成图形为主的版本，再微调文字 |

### 常见问题修正指南
| 问题 | 原因 | 修正方法 |
|------|------|----------|
| 中文出现乱码/错字 | 文字过多或字号过小 | 减少中文字数，放大字号，品牌名改用英文 |
| 右侧大面积留白 | 右侧缺少图形元素 | 在右侧描述中明确要求画具体的简笔画物体 |
| 背景出现极光/科幻感 | 模型发挥过度 | 强调"普通办公室墙面，绝对不要极光或渐变背景" |
| 插图过于精细不像简笔画 | 未强调简笔画约束 | 加入"火柴人风格""用最少线条表现""类似白板会议随手画" |
| 画面偏暗 | 默认曝光不足 | 加入"曝光补偿+1.2EV""明亮通透""日光充足" |
| 文字模糊不清 | 文字过小或过密 | 减少文字量，加大字号，每区域文字不超过2行 |
