## 一、这份日报凭什么比你手动整理的好

手动整理日报，你大概是这么干的：

打开十几个网站 → 翻公众号 → 刷微博 → 看36氪 → 复制粘贴到文档 → 排版 → 发到群里。

**花1小时，老板回你两个字：收到。**

AI日报不一样。它比你快，比你全，而且不会漏：

| | 手动整理 | AI日报 |
|---|---|---|
| 时间 | 1小时+ | 5分钟自动完成 |
| 信息源 | 你常看的那几个 | 全网13轮搜索 |
| 时效性 | 经常混入旧闻 | 锁定24小时以内 |
| 分析 | 你自己想 | 每条自带行业价值分析 |
| 频率 | 心情好才写 | 每天6点雷打不动 |

但最关键的区别不是效率——**是每条信息后面多了一段"行业价值分析"。**

不是告诉你"发生了什么"，是告诉你"跟你有什么关系、你该怎么应对"。

**这才是日报该有的样子。**

## 二、我是怎么做的

工具只需要一个：**OpenClaw**。

OpenClaw是一个AI Agent框架，能让AI自动搜索、自动执行任务、定时推送到微信/飞书/Telegram。简单说就是：**给AI排班，让它每天自动帮你干活。**

整个搭建过程就3件事：

### 1. 定义板块：你关心什么，日报就搜什么

我做的是AI行业日报，分了5个板块：

- 🔥 **模型速报** — 大模型发布、更新、评测
- 🛠️ **工具动态** — AI工具新品、重大更新
- 📰 **行业热点** — 融资、政策、重大事件
- 👤 **博主精读** — KOL深度文章、独到观点
- 🧰 **GitHub热门** — 开源项目趋势

**板块不超过5个，每个板块3-5条。** 多了没人看。

### 2. 写Prompt：搜索策略决定日报质量

这一步最重要。好的日报和垃圾日报，差距就在prompt上。

我踩了无数坑之后，总结出3条铁律：

**铁律一：必须搜，不能编。**

一开始我让AI"写一份日报"，它一本正经地编了5条假新闻，链接点进去404。我差点转发到工作群。

现在prompt最顶部写死：

> 所有信息必须来自搜索结果。搜不到就写「今日暂无」，绝对不编造。

**铁律二：多轮搜索，不要一次搜完。**

原来每板块只搜1次，经常漏。现在改成**13轮分组搜索**——模型动态搜3轮、工具应用搜3轮、热点搜2轮……全部锁定24小时以内。

**铁律三：每条必须带行业价值分析。**

原来的格式：`GPT-5.2发布 🔗链接`——看完等于没看。

现在的格式：

> GPT-5.2发布，支持原生多模态
> 📌 **行业价值**：内容创作者影响最大，图文视频一站式生成。中小团队可替代部分设计外包。
> 🔗 来源链接

**多一段分析，日报价值翻倍。** 老板看完直接能做决策。

### 3. 定时推送：每天6点自动跑

OpenClaw支持Cron定时任务。设好之后每天早上6点自动执行：

搜索 → 筛选 → 生成 → 推送到Telegram

我醒来打开手机，日报已经在那了。比闹钟还准。

## 三、电商版日报长什么样

同样的方法，换一套板块：

- 📊 **平台政策** — 淘宝/抖音/拼多多规则变动
- 🔥 **爆款速报** — 各平台24小时内跑出来的爆品
- 🛠️ **工具更新** — 千牛/抖店/电商SaaS新功能
- 💰 **行业数据** — GMV、投流ROI、行业报告
- 👤 **操盘手分享** — TOP卖家公开经验

板块换了，铁律不变：不编造、多轮搜、每条带分析。

**任何行业都一样：换板块、换关键词，日报就是你的了。**

## 通用模板：直接复制

不管你是什么行业，把下面模板里的`【】`换成你的内容就能用：

```
你是一个【行业名称】行业分析师。
每天生成一份行业日报，推送给团队。

## 板块设置
1. 【板块1名称】（搜索关键词：【关键词A/关键词B/关键词C】，3-5条）
2. 【板块2名称】（搜索关键词：【关键词A/关键词B】，3-5条）
3. 【板块3名称】（搜索关键词：【关键词A/关键词B】，2-3条）
4. 【板块4名称】（搜索关键词：【关键词A/关键词B】，2-3条）
5. 【板块5名称】（搜索关键词：【关键词A/关键词B】，2-3条）

## 铁律
- 所有信息必须来自真实搜索结果，每条附来源链接
- 只搜24小时以内的内容，过期不要
- 搜不到就写「今日暂无」，绝对不编造任何新闻、数据或链接
- 每条信息后加「📌 行业价值」：这条跟从业者有什么关系，该怎么应对
- 每板块不超过5条，不够不凑，超出只留最有价值的

## 输出格式
Markdown格式，简洁有力，每条不超过3行。
标题：【行业名称】日报 · {日期}
```

**复制 → 改关键词 → 丢给OpenClaw → 设个定时任务 → 搞定。**
