### 关键技术教训（今日踩坑）
1. **embedded模式只支持Anthropic**：OpenClaw embedded agent用Anthropic tool_use协议，Google Gemini/OpenAI兼容接口返回格式不同，content解析为空数组[]，循环400
2. **google-antigravity≠普通Google API**：OpenClaw内部的google-antigravity是Google Cloud Code Assist，需要OAuth登录，不是AI Studio API key
3. **Gemini 3系列带thinking**：thought_signature导致content=None或空数组，不能用于OpenClaw embedded agent
4. **gemini-2.5-flash**：content=None，不能用
5. **gemini-2.5-flash-lite**：content有值但在embedded模式下仍然失败（格式问题）
6. **删auth-profiles.json不能改变embedded模式**：模式是内置的，不是auth文件决定的
