# AI日报精简版模板 v2

> 设计原则：手机2-3屏、3条新闻、每条2行、有态度、去冗余
> 替代原方案A/B，基于老大反馈进一步精简

---

## 模板

```
🤖 零零壹AI日报 | {月}月{日}日 {星期}

▎{新闻1标题}
{一句话点评，说人话，有态度，≤40字}

▎{新闻2标题}
{一句话点评}

▎{新闻3标题}
{一句话点评}

💡 {今日一句：一句话洞察或金句，≤30字}
```

---

## 选稿规则

1. **只选3条**，按重要性排序，宁缺毋滥
2. 优先级：产品发布/重大更新 > 政策变动 > 融资/人事 > 论文
3. 筛选标准：**对普通AI使用者有直接影响或启发**
4. 过滤：纯论文、纯融资、纯八卦、重复旧闻一律不选

## 去掉的板块（v1有v2砍掉）

- ~~大佬动态追踪~~ → 除非真正改变格局的（如CEO离职/公司合并），否则不放
- ~~Skills资源推荐~~ → 低价值，砍
- ~~5条新闻~~ → 精简到3条
- ~~零零壹点评段落~~ → 合并到每条新闻的一句话点评里

## 点评要求

- 说人话，有观点，不要"值得关注"这种废话
- 要么说影响（"你的XX workflow要变了"），要么说判断（"雷声大雨点小"）
- 每条点评≤40字

## 今日一句

- 可以是行业金句、个人洞察、或反直觉观点
- 一行搞定，不要鸡汤
- 例："工具免费了不代表你的时间免费。"

## 格式规范

- 总字数控制在200字以内
- 不用表格、不用分隔线轰炸
- emoji克制：标题1个🤖 + 结尾1个💡，正文0个
- 竖线▎做视觉锚点，轻量不花哨
