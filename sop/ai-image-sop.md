# AI配图SOP — 白板风/手绘风

## 流程

```
① 找参考图 → ② 发给001反推提示词 → ③ Lovart生成 → ④ 微调出图
```

### ① 找参考图
- 来源：小红书、公众号、Twitter、Pinterest
- 看到好看的AI生成图直接保存
- 重点关注：风格、布局、配色、文字排版

### ② 反推提示词
- 把参考图发给001
- 001分析图片的：风格、构图、元素、配色、文字内容
- 输出一段中文提示词

### ③ Lovart.ai 生成
- 网址：https://www.lovart.ai/zh/home
- 模型：Nano Banana Pro（免费积分）
- 粘贴提示词，生成初版

### ④ 微调
- 不满意的地方告诉001，调整提示词再生成
- 常见调整：文字内容、颜色、布局、元素增减
- 一般2-3轮就能出满意的图

## 提示词模板（白板风）

```
办公室白板照片，所有文字马克笔手写体。
顶部[颜色]粗体手写：[标题]。
[内容区域描述：分几块、每块写什么、配什么简笔画]。
白板有反光光斑和擦不干净的灰色使用痕迹。
银色金属边框，底部托盘红绿蓝黑四色马克笔和板擦。
真实办公室背景，自然光线，超高清摄影风格。
```

## 关键经验
- 中文提示词比英文效果好（Lovart对中文理解更强）
- 文字越少越清晰，信息密度靠图形元素补
- 颜色限定4种（红绿蓝黑）最协调
- 加"反光""擦痕"等细节让白板更真实
- 万相2.6中文渲染不行，白板风统一用Lovart
