# DeepSeek V4今天发：它要做的事，让我把OpenAI的订阅取消了

上次DeepSeek V3发布那天，我正在睡觉。

醒来发现朋友圈全在刷屏，英伟达股价跌了17%，硅谷的人已经开始写检讨。

那天我盯着手机想了很久：这玩意跟我有什么关系？

这次V4要来了，我决定提前盯着。

> 📅 **发布预告：** TechNode 3月2日爆料本周发布，业内普遍预计3月5日-7日上线。这两天就等消息，随时准备测。

---

## 一、它是什么？先说人话

先说三个关键信息，读完这三条你就比90%的人都清楚了。

**第一：多模态。**

说人话就是：不只能聊天，还能看图、生成图片、处理视频。

这是一个门槛。以前的DeepSeek只能文字，这次直接跳到跟GPT-4o、Claude同一个段位。

**第二：国产芯片优化。**

<font color="red">**V4是在华为+寒武纪芯片上训练和优化的，不靠英伟达。**</font>

这件事的意义比模型本身还大。

美国封芯片，DeepSeek绕开了，还绕出了同等性能。

这不是「我们也做到了」，这是「你的制裁手段失效了」。

**第三：14个月没动静，这次是憋大招。**

上次V3/R1发布是2024年底，距今14个月。

DeepSeek这家公司不爱说话，但每次说话都是一声炮响。

---

## 二、对我来说，最期待的一件事

我每天用 OpenClaw 驱动 AI 工作流，说白了就是：AI替我干活，我来调度和验收。

现在最大的成本是 token 费用。

Claude Opus 跑一个稍微复杂点的任务，一次就能烧几毛到几块。

累计下来，一个月随随便便几百块。

<font color="red">**如果 DeepSeek V4 接入 OpenClaw 作为底层模型，同等能力、价格打两折——这个成本直接腰斩。**</font>

不是便宜一点点，是砍掉一大半。

对想用 AI 写代码、跑工作流、做自动化的普通人来说，这条路的门槛会低得多。

以前：想跑一个编程 Agent，光 API 费用就让你望而却步。

现在：如果 V4 开放 API 还是 DeepSeek 的定价风格（便宜到离谱），你真的可以用它来驱动一套完整的 AI 工作流。

---

## 三、它会搅动什么局？

说一个很多人没注意到的角度。

多模态这件事，之前是 OpenAI 和 Claude 的护城河。

GPT-4o 的看图能力、Claude 的分析能力，这是他们的差异化。

<font color="red">**如果 DeepSeek V4 的多模态能力达到同等水平，而且开源或低价——护城河就不存在了。**</font>

中国有一种打法：不在你的优势点硬刚，绕到后面，用价格把你的壁垒清零。

V3 就是这么赢的。

V4 如果延续这个逻辑，这次被搅动的不只是股价，而是整个 AI 工具的付费逻辑。

---

## 四、V4发布后，我会先测什么

等 V4 正式发布，我打算按这个顺序来：

**① 先接入 OpenClaw 跑一个实际任务。**

不测 demo，直接用真实工作流测。比如：让它帮我写完一篇文章草稿，看速度和质量。

**② 对比 token 消耗。**

同一个任务，V4 vs Claude，记录费用。如果 V4 便宜一半以上，值得切换。

**③ 测代码生成能力。**

丢一个我自己真实遇到的编程问题，让它给解法。看逻辑对不对、能不能跑。

**④ 测多模态（如果支持）。**

丢一张截图让它分析，或者让它生成一张图，看中文渲染质量。
