# GPT-5.4发布速报

OpenAI深夜发布GPT-5.4，三个版本：标准版、Thinking推理版、Pro高性能版。

## 核心亮点

### 1. AI学会操作电脑了
- OSWorld测试：GPT-5.4得75.0%，超过人类72.4%
- 上一代GPT-5.2只有47.3%，一代涨28个百分点
- 可以通过截图+鼠标键盘操作桌面软件

### 2. 100万token上下文
- API版支持100万token（OpenAI史上最大）
- 一整本书丢进去一次性处理

### 3. 更少胡说八道
- 单条事实错误减少33%
- 整体含错误回复降低18%

### 4. 全面超越竞品
- 代码SWE-Bench：57.7%（Gemini 54.2%）
- 知识工作GDPval：83%（Claude 78%）
- 投行建模：87.3%（上代68.4%）
- 价格：$2.50/百万输入token

### 5. 原生Computer Use
- 第一个自带桌面操作能力的通用模型
- 不再需要单独的专用模型
